はじめに

※ この自動化ツールは、健康管理報告フォームの回答を自動化するものであり、日々の検温による健康管理自体を蔑ろにするものではありませんので、ご理解の程よろしくお願い致します。またご自身の体調にはくれぐれも注意し、ご自愛くださいませ。

自動化ツールの使い方

以下の手順で自動化ツールの設定を行います。なお、全てボタン操作でできるため、プログラミング等の知識は必要ありません。

  1. github に登録する。
  2. Repository "Form-Auto-Fill-In" を Fork する。
  3. github actionsを有効化する。
  4. Fork した Repository の Secrets にフォーム回答時に必要な情報を記入する。
  5. (正しく動くかテストする。)
  6. Form-Auto-Fill-In や Translation-Gummy などの Repository に Star を付ける。（最重要です🥺🥺🥺）

※ わからないことがありましたら、 や、その他の手段でご連絡ください。
